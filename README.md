Cette application a pour objectif de gérer une application d'entrainement sportif. Il y plusieurs types d'exercices.
Pour chaque bloc, il faudrait pouvoir configurer le temps, le repos entre les exercices. Le nom des exercices ainsi que le temps écoulé doit être affiché.
Certains exercices se calcule en nombre de répétitions, d'autre en durée.
Je veux pouvoir passer à l'exercice suivant en appuyant sur un bouton.
Chaque bloc est configurable unitairement et je peux ensuite construire mon programme en choisissant les blocs que je veux ajouter.
